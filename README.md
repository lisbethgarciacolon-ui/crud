# CRUD1 - Gestor de Tareas

## 🚀 Ejecutar la aplicación

1. Instala dependencias:

```bash
npm install
```

2. Inicia el servidor (modo desarrollo con recarga automática):

```bash
npm run dev
```

3. Abre en tu navegador:

```
http://localhost:3000
```

---

## 🧩 Estructura clave

- `server/api.js`: API REST para registrar usuarios, iniciar sesión y gestionar tareas.
- `public/index.html`: Interfaz de usuario (front-end).
- `public/gestor.js`: Lógica de interacción cliente/servidor.
- `tareas.db`: Base de datos SQLite donde se guardan usuarios y tareas.
